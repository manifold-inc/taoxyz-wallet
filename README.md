# taoxyz-wallet

## Required programs

- web-ext https://github.com/mozilla/web-ext
- bun https://bun.sh/docs/installation

To install node dependencies:

```bash
bun install
```

To compile:

```
bun run build
bun run build:firefox
```

Dist will be in `./dists/firefox_zip`

## Chrome

To Run

```bash
bun run index.ts
```
